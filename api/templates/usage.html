Usage

Usage Instructions:

For optimal performance, please adhere to the following guidelines when providing input files:

- Ensure that the file contains unique filenames in a column labeled 'filename', using only ASCII characters and no special characters.

- If providing sample descriptions, prefix them with 'ATTRIBUTE_*'.

- Any subsequent columns should contain quantitative data, named in the format 'number representing mz_ number representing rt'.

- The file Must be encoded in UTF-8 read this document if you need help with encoding.

How to Open the .qzv downloaded Emperor Plot?

The .qzv file contains HTML, JavaScript, and CSS files that your browser can read. To view it, just drag and drop the file onto this page: Qiime 2 Viewer.

Storing .qzv files can be helpful for revisiting an analysis, sharing with collaborators, or including in a publication.
